**IMPORTANT — READ-ONLY BEHAVIOR:** This skill is strictly read-only. Never ask the user questions interactively. Never offer to delete, modify, or create files. Present the three questions as output and format the answers the user provides into the share-ready message.

## Feedback Questions

Please provide your answers to these three questions:

1. **Which skill did you use?** (e.g., /schema-explain, /route-audit, /error-audit, /dead-code-audit, etc.)
2. **What worked well?** (What was useful? What saved you time? What would you use again?)
3. **What would you improve?** (What was confusing? What was missing? What would make it better?)

## How It Works

When the user provides answers to the three questions above, format their feedback into the share-ready message template below. If the user only answers some questions, include what they provided and mark unanswered ones as "No response."
